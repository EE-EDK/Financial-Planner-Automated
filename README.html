üíªUsing the Program

Double-click finance_gui.exe to open the program.

The program has 2 tabs:

Operations Tab

  • IMPORT - Load your transactions
  • ANALYZE - Check your spending
  • REPORT - Create the report
  • RUN ALL - Do everything at once (easiest!)

Budget Editor Tab

  • Edit your monthly income
  • Change budget amounts for each category
  • Click "SAVE CONFIG" when done

üÜòHelp! Something's Wrong

"Config not found"

Make sure config.json is in the tools folder

"No transactions found"

  • Make sure you dropped your Excel file in data/inputs, OR
  • Make sure you saved transactions.csv in the data folder
  • Check that CSV is actually a CSV file (not Excel)

"The program won't run"

  • Try restarting your computer
  • Make sure the .exe file is in the right folder
